# Rank emblems

Put your rank emblem PNG files in this folder so the frontend can load them reliably:

- `iron.png`
- `bronze.png`
- `silver.png`
- `gold.png`
- `platinum.png`
- `emerald.png`
- `diamond.png`
- `master.png`

Notes:
- This project intentionally does **not** hotlink third-party images.
- The original reference URLs were returning a Cloudflare `403 challenge`, so they won't load consistently.
